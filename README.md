# 🍀Ureca 3기 백엔드 대면반 CS 스터디 

## 📡Network

### 1. 스터디 소개

> 네트워크에 대한 기본 개념부터 실제 인터넷이 동작하는 방식까지 깊이 있게 이해하며 CS 면접, 개발 실무, 시스템 설계에 활용할 수 있도록 합니다.
> 매주 스터디마다 1인당 3개의 면접 대비 질문을 준비하여 서로 질문하고 답변하는 스터디입니다.


</br>


### 2. 진행 방식
- 주 1회 (목요일)
- 각 회차별 진행
  - 챕터별 정리 + 도식화
  - 1인당 면접 질문 3개 준비
  - 질문마다 답변 진행 
  - 매주 질문 / 답변 폴더에 정리
    
 📌 저장소 활용 방식
  - 저장소 clone -> 스터디 전 Pull Request -> 스터디 종료 후 Pull Request 병합 진행
  - `/week-00` : 주차
    - `/이름` : 개인 별 내용 정리
  - `/QnA` : 질문과 답변
    - `/week-00` : 주차
